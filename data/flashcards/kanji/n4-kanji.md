# JLPT N4 Kanji
#
# ABOUT THIS SET:
# This set contains all kanji required for JLPT N4 level (approximately 220 new kanji beyond N5).
# N4 kanji build upon N5 foundations and include characters for everyday situations,
# academic contexts, work environments, and more complex compound words.
#
# TOTAL N4 KANJI: ~220 (plus 80 from N5 = 300 total through N4)
#
# ORGANIZATION:
# Kanji are presented with kun'yomi (訓読み) and on'yomi (音読み) readings.
# Each entry includes common vocabulary words, stroke counts, and learning tips.
#
# READING CONVENTIONS:
# - Hiragana = kun'yomi (native Japanese reading)
# - Katakana = on'yomi (Chinese-derived reading)
#
# NOTE: This set contains ONLY the new kanji introduced at N4 level.
# For N5 kanji, refer to the separate N5 Kanji set.

## 泳

**Reading:** およ-ぐ / エイ
**Meaning:** swim
**Type:** kanji
**Level:** N4
**Tags:** #jlpt-n4, #kanji, #verb, #sports

### Example Words
- 泳ぐ (およぐ) - to swim
- 水泳 (すいえい) - swimming
- 泳ぎ方 (およぎかた) - way of swimming
- 平泳ぎ (ひらおよぎ) - breaststroke

### Notes
Stroke count: 8. Radical: 水 (water). Shows water (氵) + eternal (永), suggesting continuous movement through water. Remember: "Swimming eternally in water."
